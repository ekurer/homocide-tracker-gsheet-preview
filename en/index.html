Homicide Victim Tracker in Arab Society in Israel

This searchable landing page points to an interactive dashboard that organizes homicide victim records in Arab society in Israel by year, locality, gender, weapon type, and solve status.

Open dashboard Victim names Code and data

What the site includes

  • A map by locality of residence rather than incident point.
  • Year-over-year comparison at the locality level.
  • Gender, weapon, and indictment indicators.
  • Dedicated Ramadan analysis views.
  • A raw victim table with source links.

What searches this page is meant to serve

This page is designed to rank for searches such as "Arab society homicide tracker", "homicide victims in Arab society 2026", "murders by locality in Israel", or "victim names Arab society Israel".

Data coverage

The current dataset spans 2018 through 2026 and is available both as an interactive dashboard and as downloadable JSON and CSV files.

Methodology

The map places each record at the centroid of the victim's locality of residence. It does not attempt to represent exact homicide coordinates.

Quick navigation

Where to start

The main dashboard is best for an overall view. The comparison screen is for year-over-year locality analysis. The victim table is useful for reporters, researchers, and advocates who need names and source links quickly.
